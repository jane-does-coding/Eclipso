## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/jane-does-coding/eclipso.git
   ```
2. Navigate to the project directory:
   ```sh
   cd eclipso
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the development server:
   ```sh
   npm run dev
   ```
